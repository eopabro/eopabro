# Foco de Pesquisa
- Modelagem de séries temporais para dados financeiros <br>
- Engenharia de atributos e lojas de atributos <br>
- Machine Learning aplicado a sinais financeiros <br>
- Rastreamento de experimentos e reprodutibilidade <br>

- Subindo sistemas de pesquisa para produção

---

# Repositórios Principais

market-data-pipeline: ingestão, limpeza e armazenamento de dados financeiros em Parquet com DuckDB.

quant-feature-engineering: construção e versionamento de sinais quantitativos e estatísticas derivadas.

financial-time-series-models: implementação de modelos AR, ARIMA e GARCH para previsão de séries temporais.

ml-trading-models: modelos de ML com XGBoost/LightGBM e validação temporal rigorosa.

backtesting-engine: simulação de estratégias com custos, slippage e métricas de risco.

quant-research-lab: experimentos quantitativos, análise de fatores e notebooks de pesquisa.

ml-experiment-tracker: organização de experimentos utilizando MLflow e Weights & Biases.

quant-system: integração completa com FastAPI, Docker Compose e deploy local.
